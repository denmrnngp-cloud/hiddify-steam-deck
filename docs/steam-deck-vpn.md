# VPN for Steam Deck: Game Mode And Desktop Mode

Hiddify for Steam Deck is a Steam Deck VPN setup for SteamOS users who need internet access through VPN profiles in both Gaming Mode and Desktop Mode.

It is useful when internet access is blocked, filtered, or unstable in your country, ISP, hotel, campus, or public Wi-Fi network, and you want your Steam Deck to keep working through a VPN without switching out of Game Mode.

## What Problem This Solves

When internet access is restricted, a Steam Deck user usually needs two things:

- Desktop Mode VPN access for setup, browser use, downloads, launchers, and regular Linux desktop tasks.
- Game Mode VPN access without leaving the Steam Deck quick access menu.

This project solves that by combining:

- Hiddify VPN for the desktop client and profile management.
- Decky Loader integration for Game Mode VPN control.
- sing-box based proxy/VPN support for Hiddify profiles.
- Profile switching directly inside the Decky plugin.
- Server selection inside the Decky plugin when a profile supports multiple servers.

## Supported Steam Deck VPN Use Cases

- Steam Deck VPN in Game Mode.
- Steam Deck VPN in Desktop Mode.
- SteamOS VPN with Hiddify profiles.
- Decky VPN plugin for quick connect and disconnect.
- Game Mode server selection for multi-server VPN profiles.
- VLESS, Reality, VMess, Trojan, Shadowsocks, Hysteria2, and TUIC profiles through Hiddify/sing-box.
